Type Peptide

Type Peptide es una herramienta pedagógica para el diseño y optimización de péptidos antimicrobianos basados en sus propiedades fisicoquímicas.

La aplicación ha sido desarrollada por la línea de interacciones moleculares del grupo Biología Funcional, asociado a la Facultad de Ciencias de la Universidad Nacional de Colombia – Sede Medellín.

A partir de una cadena de aminoácidos, la aplicación calcula algunas de sus propiedades fisicoquímicas y genera una representación gráfica de una α-hélice teórica.

Cómo usarla

La aplicación está compuesta por 3 secciones:

1. Ingreso de datos

Ubicada en la parte izquierda. Para iniciar a utilizar la aplicación, ingrese caracteres en el campo “Escribir aquí la secuencia”. Solo se reciben caracteres que correspondan al código de una letra de los 20 aminoácidos más comunes.

2. Representación gráfica

Ubicada a la derecha; muestra la cadena peptídica como una α-hélice.
Para ver la cadena lateral y su posición, haga clic sobre el aminoácido (clic de nuevo para ocultar).
Para manejar la cámara use clic izquierdo/derecho y arrastre para rotar o desplazar.

3. Información

Se calculan características del péptido ingresado y se muestran en la tabla.
